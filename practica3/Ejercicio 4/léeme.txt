Lenguaje de programación: C#
Versión del compilador:  .NET Framework 4.6.1

En el directorio de Ejercicio 4 están la carpeta que incluye el código fuente y el ejecutable "Json2xml.exe", que al ejecutarlo pedirá un archivo JSON de entrada que convertirá a xml. Como output se creará un archivo de tipo xml con el mismo nombre que tenía el archivo de entrada.

Archivos JSON utilizados de prueba (los que dieron resultado en el ejercicio anterior):
 - rutas.json
 - books.json

Archivos generados:
 - rutas.xml
 - books.xml
